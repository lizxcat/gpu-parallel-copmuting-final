# Final Project – GPU Acceleration with CUDA

## Project Overview

This project demonstrates GPU acceleration using CUDA for parallel processing and benchmarking against CPU execution.

The goal of the project was to compare execution times between CPU and GPU implementations and analyze the performance improvements achieved through parallel computing.

---

## Features

- CPU implementation
- CUDA GPU implementation
- Benchmarking and timing comparison
- Performance analysis

---

## Requirements

### Hardware
- NVIDIA GPU with CUDA support

### Software
- CUDA Toolkit
- C++ Compiler
- Windows/Linux environment

---

## Required Packages / Dependencies

- CUDA Toolkit
- NVIDIA Drivers
- Standard C++ libraries

---

python final_demo.py
